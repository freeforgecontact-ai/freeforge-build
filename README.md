# FreeForge Apps — 100% local, hors-ligne, open source

Suite d'outils PME et personnels **100% locaux** (par **PGRG**). 9 applications groupées (~90 outils) + **VibeLocal** (musique hors-ligne), un moteur vanilla partagé, l'identité visuelle ia.pgrg.ca.

- **Installeurs natifs** (Android `.apk`, Windows `.exe`, macOS `.dmg`, Linux `.AppImage`) : voir l'onglet **Releases** ou les billets sur freeforge.pgrg.ca.
- **100% hors-ligne** : tout fonctionne sans connexion. Aucune donnée envoyée sur un serveur.
- **Open source (MIT)** : le code de chaque outil est ici, lisible et réutilisable.

Compilation automatique via GitHub Actions (Capacitor pour Android, Electron pour le bureau).
